📋 RESUMO: Esta política explica como coletamos, usamos, armazenamos e protegemos seus dados pessoais, em conformidade com a LGPD (Lei Geral de Proteção de Dados - Lei 13.709/2018).

1. DADOS COLETADOS

1.1. Dados de Cadastro (Obrigatórios)

  • Nome completo
  • Endereço de email
  • Número de telefone/WhatsApp
  • CPF
  • Profissão
  • Anos de experiência

1.2. Dados Opcionais

  • Foto de perfil
  • Habilidades profissionais
  • Fotos de serviços/portfólio

1.3. Dados de Uso

  • Vagas publicadas
  • Candidaturas realizadas
  • Mensagens do chat
  • Avaliações enviadas/recebidas
  • Conexões de rede
  • Localização (para funcionalidade do mapa)

2. FINALIDADE DO USO

  • Criar e gerenciar sua conta
  • Conectar profissionais com empreiteiros
  • Exibir seu perfil profissional
  • Enviar notificações de vagas e mensagens
  • Melhorar a experiência do usuário
  • Cumprir obrigações legais

3. COMPARTILHAMENTO DE DADOS

✅ NÃO VENDEMOS SEUS DADOS!
  • Nome e profissão: Visíveis publicamente no perfil
  • WhatsApp: Visível para usuários logados
  • CPF: NUNCA é visível para outros usuários
  • Email: Visível para usuários logados
  • Mensagens: Privadas entre os participantes
  • Localização: Usada apenas para o mapa

4. ARMAZENAMENTO E SEGURANÇA

  • Servidores: Firebase (Google Cloud)
  • Criptografia: HTTPS/SSL em trânsito
  • Senhas: Hash criptografado (Firebase Auth)
  • Backup: Realizado automaticamente
  • Acesso: Restrito a administradores autorizados

5. SEUS DIREITOS (LGPD)

  • ✅ Acessar seus dados
  • ✅ Corrigir dados incompletos
  • ✅ Solicitar exclusão total
  • ✅ Revogar consentimento
  • ✅ Portabilidade dos dados
  • ✅ Saber com quem compartilhamos

6. EXCLUSÃO DE DADOS

Para excluir sua conta e todos os dados:

  1. Acesse o app e vá em "Meu Perfil"
  2. Clique em "SAIR"
  3. Envie um email para: lpxdesenvolvimento@gmail.com
  4. Assunto: "EXCLUSÃO DE DADOS - [Seu Nome]"
  5. Seus dados serão excluídos em até 7 dias úteis

7. COOKIES E TECNOLOGIAS

  • Usamos armazenamento local para melhorar a experiência
  • Não usamos cookies de rastreamento de terceiros
  • Firebase Analytics (dados anônimos de uso)
