1. INFORMAÇÕES COLETADAS

1.1. Dados de Cadastro

  • Nome completo
  • Endereço de email
  • Número de telefone (WhatsApp)
  • CPF
  • Profissão e experiência
  • Habilidades profissionais
  • Foto de perfil (opcional)

1.2. Dados de Uso

  • Vagas publicadas e candidaturas
  • Mensagens trocadas no chat
  • Avaliações recebidas e enviadas
  • Conexões de rede
  • Localização (para funcionalidade do mapa)

2. USO DAS INFORMAÇÕES

Seus dados são utilizados para:

  • Criar e gerenciar sua conta
  • Conectar profissionais e empreiteiros
  • Exibir seu perfil para outros usuários
  • Enviar notificações relevantes
  • Melhorar nossos serviços

3. COMPARTILHAMENTO DE DADOS

  • NÃO vendemos seus dados pessoais
  • Seu nome, profissão e avaliações são visíveis publicamente
  • Seu WhatsApp é visível para usuários logados
  • Seu CPF NÃO é visível para outros usuários
  • Mensagens do chat são privadas entre os participantes

4. ARMAZENAMENTO E SEGURANÇA

  • Dados armazenados no Firebase (Google Cloud)
  • Criptografia em trânsito (HTTPS/SSL)
  • Senhas armazenadas com hash (Firebase Auth)
  • Acesso restrito a dados sensíveis

5. SEUS DIREITOS

  • Acessar seus dados a qualquer momento
  • Corrigir informações incorretas
  • Solicitar exclusão da conta e dados
  • Revogar consentimento de uso

6. EXCLUSÃO DE DADOS

Para excluir sua conta e todos os dados associados, entre em contato pelo email: suporte@lpxconstrutor.com.br
